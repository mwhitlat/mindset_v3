Troubleshooting

  • No badge visible? Make sure the Mindset extension is installed and enabled
  • Badge not updating? Check the browser console for error messages
  • Wrong colors? Try refreshing the extension in chrome://extensions/
  • Still not working? Try disabling and re-enabling the extension
