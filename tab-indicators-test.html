What are Tab Indicators?

Tab indicators are small colored circles that appear on each webpage showing:

  • Content Type - News, social, entertainment, educational, etc.
  • Credibility Level - Color-coded (green=high, orange=medium, red=low)
  • Content Tone - Emoji indicators for uplifting vs cynical content

Look for the small colored circle in the top-right corner of this page!

Indicator Features

  • Hover Effect - Hover over the indicator to see detailed tooltip
  • Pulse Animation - Low credibility sources pulse to draw attention
  • Real-time Updates - Indicators update as you navigate between pages
  • Non-intrusive - Small, subtle, doesn't interfere with browsing
  • Customizable - Can be enabled/disabled in settings

How It Works

  1. Page Analysis - Extension analyzes each page for content type and credibility
  2. Visual Indicator - Small colored circle appears in top-right corner
  3. Real-time Updates - Indicator changes as you browse different sites
  4. Tooltip Information - Hover to see detailed breakdown
